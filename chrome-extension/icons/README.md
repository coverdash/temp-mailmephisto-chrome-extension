# Icon Generation

## Quick Method: Using the HTML Generator

1. Open `generate-icons.html` in your web browser
2. Click each "Download" button to save the icons
3. The icons will be saved as PNG files in your downloads folder
4. Move them to this `icons/` directory

## Alternative: Create Custom Icons

You can create your own icons using any graphics editor (Photoshop, Figma, GIMP, etc.):

**Required sizes:**
- `icon16.png` - 16x16 pixels (toolbar icon)
- `icon32.png` - 32x32 pixels (retina toolbar icon)
- `icon48.png` - 48x48 pixels (extension management)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

**Design Guidelines:**
- Use a simple, recognizable symbol (email/envelope icon recommended)
- Ensure good contrast for visibility
- Test at all sizes for clarity
- Use PNG format with transparency
- Maintain consistent style across all sizes

## Design Concept

The default icons feature:
- Red-to-orange gradient background (#ef4444 to #f97316)
- White envelope icon
- Rounded corners for modern look
- Clear visibility at all sizes

## Manual Icon Creation Steps

If you prefer to create icons manually:

1. Create a 128x128 canvas in your preferred graphics tool
2. Add a gradient background (red to orange)
3. Draw a simple envelope icon in white
4. Export as PNG
5. Resize to 48x48, 32x32, and 16x16, exporting each size

Make sure to optimize for clarity at smaller sizes!
